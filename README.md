Deskripsi Proyek
Aplikasi Todo List adalah REST API berbasis FastAPI yang digunakan untuk manajemen tugas pribadi dengan sistem autentikasi pengguna menggunakan JWT.
Proyek ini dikembangkan sebagai Project Akhir Mata Kuliah dengan tujuan menerapkan konsep backend modern, keamanan API, dan arsitektur berlapis.

Tujuan Pengembangan
Mengimplementasikan REST API menggunakan FastAPI
Menerapkan autentikasi dan otorisasi berbasis JWT
Menggunakan MySQL sebagai basis data relasional
Menerapkan clean layered architecture (schema, repository, service, router)
Melatih kerja tim dan version control menggunakan GitHub

Registrasi pengguna

Manajemen Tugas
Menambahkan tugas
Menampilkan daftar tugas
Mengubah tugas
Menghapus tugas
Filter dan pencarian tugas (opsional)

Desain Database
Database menggunakan MySQL
Relasi One-to-Many antara User dan Todo
Struktur tabel dan contoh data disediakan dalam file SQL

Dokumentasi API
FastAPI menyediakan dokumentasi otomatis:
Swagger UI: /docs
ReDoc: /redoc
Dokumentasi ini menampilkan seluruh endpoint, parameter, dan contoh request/response.
